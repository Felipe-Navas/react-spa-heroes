## Getting Started

### Dependencies

- `npm`
- `node`

### Installing

- Clone the repository and install dependencies:

```
git clone https://github.com/Felipe-Navas/react-spa-heroes.git && cd react-spa-heroes && npm install
```

### Executing program

```
npm start
```
